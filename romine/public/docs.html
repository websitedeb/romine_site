Bot Control Panel Documentation

This documentation provides an overview of the UI components and functionality of the Bot Control Panel, which is built using Python's Tkinter and Selenium.

UI Components

Labels

The application has several labels to display information:

  • Count: Displays the number of searches performed.
  • Point(s): Displays the current points.
  • Should Increment by: Displays the number of points that should have been added during the session.

Entry

The application includes an entry field for setting the wait time between searches:

  • Wait Time (seconds): Accepts the wait time in seconds (default is 4.5 seconds).

Buttons

The application includes two buttons:

  • Start Bot: Starts the bot. This button initiates the search process using a separate thread.
  • Stop Bot : Stops the bot. This button sets a flag to stop the search process.

Functionality

Starting the Bot

When the Start Bot button is pressed, the bot starts performing searches based on randomly generated words.

Stopping the Bot

When the Stop Bot button is pressed, the bot stops performing searches.

Updating Labels

The application updates the labels to display the latest search count, current points, and points added during the session.
